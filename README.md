🥤 Coca-Cola Brand Landing Page Clone

📌 Overview

This project is a static brand landing page inspired by the Coca-Cola website.
It was built using HTML and CSS to practice brand-focused UI replication and layout structuring.

The project does not include dynamic functionality.

Brand-focused design

Color scheme replication

Hero section

Promotional content sections

Responsive layout
